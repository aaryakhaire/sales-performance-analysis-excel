# Sales Performance Analysis Using Excel

## Project Overview
This project analyzes sales data to evaluate regional and product-wise performance using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

## Analysis Performed
- Region-wise sales analysis
- Product-wise revenue comparison
- Average sales value analysis

## Key Insights
- The North region contributes the highest overall sales
- Laptops generate the maximum revenue among all products
- Mobile and tablet sales remain consistent but lower than laptops
- High-value products significantly impact total revenue
